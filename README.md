# DOAN

## 프로젝트 개요

**목표**: 사용자가 간단한 입력으로 뜨개질 도안을 시각적으로 생성하고, PDF 형식으로 출력할 수 있는 웹 기반 도안 에디터 개발

**핵심 가치**:

- 한국인이 쉽게 도안을 만들 수 있는 직관적인 UI
- 전문가도 세부적으로 커스터마이징 가능한 유연성
- 국제표준 뜨개 기호를 사용한 차트 생성
- 도안을 저장하고 공유할 수 있는 기능

## 주요 기능 (MVP)

**도안 생성**

- 격자 설정 시 도안의 기본 모양 생성
- 기호 또는 색상 선택 후 마우스 혹은 터치로 격자에 기호나 색상이 칠해짐
- 격자 위에 색상 패턴을 추가할 수 있음
- 격자 위에 기호 패턴을 추가할 수 있음
- 색상 격자 위에 기호 패턴을 추가할 수 있음
- 대칭 기능을 추가해서 가로 또는 세로 대칭을 추가할 경우 해당 방향으로 도안이 대칭되어 생성
- 점대칭(회전 대칭): 중심점 기준 180° 회전 대칭 모드 지원 — 숄 칼라, 레이스, 원형 요크 등에 활용
- 영역 선택 & 복사/붙여넣기: 직사각형 영역을 드래그로 선택하고 Ctrl+C / Ctrl+V로 복사·붙여넣기 — 반복 무늬를 여러 위치에 빠르게 배치
- 단 반복 블록 마커: 반복할 행(row) 범위를 지정하고 "N회 반복" 레이블 부여 — PDF 출력 시 반복 구간 자동 표기
- 형태선(Shape Guide) 오버레이: 목선·진동·소매산 등 의류 실루엣을 격자 위에 반투명 가이드라인으로 표시 — 셰이핑 위치를 직관적으로 확인하며 기호 배치
- 격자 설정, 마우스 클릭 등 동작이 있어날때마다 실시간 도안 미리보기 제공

**도안 저장**

- 생성한 도안 저장
- pdf 형식으로 저장
- 도안의 상단에 도안 제목과 난이도 출력
- 도안 난이도는 ⭐️ 1개부터 5개로 나눔
- mvp는 DB 없이 로컬스토리지에 저장 (최대 1개)
- 이 도안에 필요한 준비물 작성 및 첨부
- 뜨개 기호 범례 자동 생성 첨부

**도안 수정**

- 저장한 도안 수정
